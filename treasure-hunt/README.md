# 🎯 보물찾기 시스템

순차적인 게임 진행과 스토리라인을 가진 보물찾기 시스템입니다.

## 📁 폴더 구조

```
treasure-hunt/
├── index.html              # 보물찾기 진행 페이지
├── admin.html              # 관리자 페이지
├── js/                     # JavaScript 모듈
│   ├── core.js            # 핵심 로직 (프리셋, 진행 상황 관리)
│   ├── admin.js           # 관리자 페이지 로직
│   ├── default-presets.js # 기본 프리셋 (해적, 우주, 마법 등)
│   ├── ui-effects.js      # 파티클, Confetti, 시각 효과
│   ├── sound-manager.js   # 사운드 효과 시스템
│   └── animations.js      # 애니메이션 로직
├── css/                    # 스타일시트
│   ├── main.css           # 메인 스타일
│   ├── animations.css     # 애니메이션 정의
│   └── effects.css        # 시각 효과 스타일
└── README.md              # 이 파일
```

## 🚀 시작하기

### 1. 보물찾기 만들기

1. `admin.html`을 브라우저에서 엽니다
2. "📦 기본 프리셋 로드" 버튼을 클릭하여 5가지 테마 불러오기
3. 원하는 테마를 선택하고 "✅ 활성화" 클릭

또는 직접 만들기:
- "➕ 새 보물찾기 만들기" 클릭
- 게임 추가 및 순서 조정 (드래그앤드롭)
- 각 게임별 스토리, 힌트, 비밀번호 입력
- 저장 후 활성화

### 2. 보물찾기 시작

1. `index.html`을 브라우저에서 엽니다
2. 활성화된 보물찾기가 표시됩니다
3. 게임을 순차적으로 완료하세요!

## ✨ 주요 기능

### UI 효과
- **Confetti 효과**: 게임 완료 시 색종이 파티클
- **배경 파티클**: 테마별 동적 배경 (별, 마법 먼지 등)
- **카운트다운**: 게임 시작 전 3-2-1 카운트다운
- **보물 지도**: 진행 상황을 보물 지도 스타일로 표시
- **반짝이는 효과**: 잠금 해제, 완료 시 시각 효과

### 애니메이션
- **게임 카드**: 펄스, 글로우, 흔들림 효과
- **진행 바**: 부드러운 애니메이션
- **텍스트**: 타이핑 효과 (옵션)
- **전환**: Fade-in, Slide, Bounce 등

### 사운드
- **효과음**: 클릭, 성공, 잠김, 카운트다운
- **팡파르**: 최종 완료 시
- **토글**: 우측 상단 사운드 on/off 버튼

### 인터랙션
- **반응형**: 모바일 최적화
- **터치 피드백**: Ripple 효과
- **햅틱**: 진동 피드백 (지원 기기)
- **툴팁**: 잠긴 게임 호버 시 안내

## 🎨 테마

5가지 기본 테마 제공:
- 🏴‍☠️ **해적 보물**: 바다, 보물섬, 해적 모험
- 🚀 **우주 탐험**: 별, 행성, 우주 여행
- 🔮 **마법사**: 마법 학교, 주문, 시험
- 🌴 **정글 탐험**: 정글, 고대 신전, 탐험
- 🕵️ **비밀 요원**: 스파이, 미션, 첩보

## 🎮 게임 진행 방식

1. **순차적 진행**: 게임을 순서대로 완료
2. **잠금 시스템**: 이전 게임 완료해야 다음 게임 활성화
3. **진행 추적**: 실시간 진행 상황 표시
4. **힌트 시스템**: 각 게임 완료 시 다음 위치 힌트
5. **최종 보상**: 모든 게임 완료 시 특별 메시지

## 🛠️ 커스터마이징

### 새 프리셋 만들기

```javascript
// admin.html에서 직접 생성 가능
{
  name: '나만의 보물찾기',
  theme: 'pirate',  // pirate, space, magic, jungle, spy, custom
  description: '설명...',
  games: [
    {
      gameId: 'game03',  // 게임 ID
      storyText: '스토리 텍스트...',
      hintMessage: '다음 위치 힌트...',
      secretCode: '비밀번호',
      successMessage: '성공 메시지...'

    // ... 더 많은 게임
  ],

    secretCode: '최종 비밀번호'
```

## 🔧 기술 스택

- **Vanilla JavaScript**: 프레임워크 없음
- **Canvas API**: 파티클 효과
- **Web Audio API**: 사운드 효과
- **CSS Animations**: 부드러운 전환
- **LocalStorage**: 진행 상황 저장

## 📊 데이터 저장

모든 데이터는 브라우저의 `localStorage`에 저장:
- `treasureHunt_presets`: 프리셋 목록
- `treasureHunt_activePresetId`: 활성 프리셋
- `treasureHunt_progress`: 진행 상황
- `treasureHunt_soundSettings`: 사운드 설정

## 🐛 문제 해결

### 파티클이 보이지 않음
- 브라우저가 Canvas를 지원하는지 확인
- 개발자 도구 콘솔에서 오류 확인

### 사운드가 들리지 않음
- 우측 상단 사운드 버튼 확인
- 브라우저 음소거 설정 확인
- Web Audio API 지원 브라우저 사용

### 진행 상황이 저장되지 않음
- 시크릿 모드가 아닌지 확인
- 브라우저 쿠키/LocalStorage 설정 확인
