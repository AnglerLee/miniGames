# 07-Math-Race 리팩토링 완료 가이드

07-Math-Race 게임을 사용자 요청에 맞춰 대대적으로 리팩토링했습니다. 이제 게임 내에서 난이도를 선택하지 않고, 관리자가 설정한 난이도와 시간으로 게임이 진행됩니다.

## 주요 변경 사항

### 1. 설정 방식 간소화 (Admin)
- **단일 난이도 고정**: 관리자 페이지에서 `Level 1` ~ `Level 5` 중 하나를 선택하여 고정합니다.
- **단일 시간 설정**: 복잡한 레벨별 시간 설정 대신, 하나의 통합된 **제한 시간**을 설정합니다.
- **UI 변경**: `<select>` 박스와 하나의 숫자 입력 필드로 설정 UI를 단순화했습니다.

### 2. 게임 UI 단순화 (Minimalism)
- **레이아웃 최적화**: 05-Sliding-Puzzle의 레이아웃을 참조하여 **최대 너비를 500px로 제한**하고, 요소 간 비율을 조정했습니다.
- **타이머 개선**: 단순 초 단위(60)가 아닌 **분:초(01:00) 형식**으로 표시하여 가독성을 높였습니다. 폰트 사이즈(3.5rem)와 여백을 최적화하여 집중도를 높였습니다.
- **문제 카드**: 화면의 주인공이 되도록 크기를 키우고, **너비를 고정**하여 문제 길이(자리수)에 따라 흔들리지 않게 했습니다. `5 + 3` 처럼 `= ?` 없이 깔끔하게 표시됩니다.
- **진행 바**: 게임 화면 최하단으로 이동하여 시선을 방해하지 않도록 했습니다.
- **정보 제거**: 불필요한 정보(난이도, 콤보 등)를 모두 제거하여 심플함을 극대화했습니다.

### 3. 게임 로직 변경
- 게임 진입 시 관리자 설정(난이도, 시간)을 즉시 로드하여 적용합니다.
- 사용자는 복잡한 선택 과정 없이 바로 게임에 몰입할 수 있습니다.
- 문제 풀이 성공/실패 로직 및 동적 재도전(+5초) 로직은 유지됩니다.

## 검증 방법

1.  **관리자 설정 확인**:
    - `admin.html`에서 난이도를 `Lv.5 (사칙연산)`, 시간을 `120`초로 설정하고 저장합니다.
    - 게임 진입 시 타이머가 `02:00`으로 표시되는지 확인합니다.

2.  **UI 확인 (모바일 뷰 권장)**:
    - 타이머가 `MM:SS` 형식으로 잘 카운트다운 되는지 확인합니다.
    - 게임 영역 좌우 여백이 적절한지(최대 500px 중앙 정렬) 확인합니다.
    - 문제 카드에 `= ?` 없이 숫자가 깔끔하게 나오는지 확인합니다.

3.  **플레이 테스트**:
    - 정답을 맞췄을 때 다음 문제로 잘 넘어가는지, 시간 종료 시 모달이 뜨는지 확인합니다.
