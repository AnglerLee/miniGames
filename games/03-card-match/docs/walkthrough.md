# 03-card-match 개선 완료 가이드

03-card-match 게임의 기능 개선이 완료되었습니다. 난이도 조절이 관리자 페이지로 통합되었으며, 제한 시간 타이머 및 동적 재도전 로직이 적용되었습니다. 또한, UI/UX가 대폭 개선되어 더욱 몰입감 있는 플레이가 가능합니다.

## 주요 변경 사항

### 1. UI/UX 및 디자인 개선
- **디자인 (New!)**: 주 타겟층을 고려하여 카드 뒷면을 **심플하고 귀여운 큐트 스타일 패턴** 이미지로 변경했습니다. 카드 내부의 과일 이모지 크기도 대폭 확대하여 시인성을 높였습니다.
- **모바일 최적화 (New!)**: 모바일 화면에서도 게임이 한눈에 들어오도록 불필요한 여백을 줄이고, 타이머 크기를 적절히 조절했습니다. 카드 그리드 또한 화면 크기에 맞춰 자동으로 최적화됩니다.
- **타이머 중앙 정렬**: 게임 상단의 타이머를 중앙에 배치하여 가독성을 높였습니다.
- **커스텀 모달 적용**: 브라우저 기본 팝업 대신 세련된 커스텀 모달창을 적용했습니다.

### 2. 관리자 설정 페이지 (`admin.html`)
- **위치**: `games/03-card-match/admin.html`
- **기능**:
    - **카드 쌍 개수 설정**: 슬라이더를 통해 6쌍(12장) ~ 30쌍(60장)까지 설정 가능합니다.
    - **제한 시간 설정**: 게임의 제한 시간을 초 단위로 설정할 수 있습니다.
    - 설정은 `localStorage`에 저장되어 유지됩니다.

### 3. 게임 로직 개선 (`game.js`)
- **게임 흐름**: 카운트다운 타이머가 작동하며, 시간 초과 시 '재도전'을 선택하면 **제한 시간이 1초 늘어난 상태**로 재시작합니다.
- **매칭 효과**: 짝을 맞춘 카드는 화면에서 사라집니다 (투명 처리).
- **정보 표시 최소화**: 설정 정보 등 불필요한 텍스트를 제거하여 깔끔한 화면을 제공합니다.

## 검증 방법

1. **디자인 확인**
   - 게임 시작 시 카드 뒷면이 귀여운 패턴 이미지로 나오는지 확인합니다.
   - 카드 속 과일 이모지가 카드 영역을 꽉 채울 만큼 큼직하게 나오는지 확인합니다.
   - 모바일 사이즈(좁은 화면)로 창을 줄였을 때 게임 화면이 잘리지 않고 한눈에 들어오는지 확인합니다.

2. **게임 플레이 테스트**
   - 카드를 뒤집어 짝을 맞추면 카드가 사라지는지 확인합니다.
   - 시간 초과 시 나타나는 '시간 초과' 모달창에서 '재도전'을 눌러 시간이 1초 늘어나는지 확인합니다.
   - 상단 우측 '재도전' 버튼 클릭 시 초기화 여부를 확인합니다.
