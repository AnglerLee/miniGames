# 08-Word-Search 개선 사항 (Walkthrough)

게임 개발 가이드에 맞춰 관리자 설정을 강화하고 게임 로직을 개선했습니다.

## 주요 변경 사항

### 1. 관리자 설정 페이지 (`admin.html`, `admin.js`)
- **난이도 프리셋 추가**: 쉬움, 보통, 어려움을 선택하여 격자 크기와 제한 시간을 자동으로 설정할 수 있습니다.
- **세부 설정 기능**:
    - **격자 크기**: 6x6 ~ 12x12 조절 가능.
    - **제한 시간**: 게임 시간 조절 가능 (기본 180초).
    - **테마 선택**: 동물, 과일, 색깔 등 나오는 단어 종류 선택 가능.
- **UI 개선**: 슬라이더와 드롭다운을 사용하여 직관적인 설정이 가능하도록 개선했습니다.

### 2. 게임 로직 및 UI (`game.js`, `index.html`)
- **카운트다운 타이머**: 제한 시간이 줄어드는 방식으로 변경되었습니다.
- **동적 격자**: 설정된 크기에 따라 격자 칸 수가 변합니다.
- **재시도 시스템**: 시간 초과로 실패 시, `30초`를 추가하여 이어서 할 수 있는 기능을 추가했습니다.
- **모달 적용**: 기본 `alert` 창을 제거하고 `common.js`의 세련된 모달(`showSuccessScreen`, `showFailScreen`)을 적용했습니다.

### 3. 내부 구조 리팩토링
- **설정 연동**: `localStorage`에 저장된 `treasureHunt_gameConfigs` 값을 로드하여 게임에 반영합니다.
- **공통 유틸리티 사용**: 타이머, 모달 등 공통 코드를 활용하여 유지보수성을 높였습니다.

## 검증 방법 (Verification)

1.  **관리자 페이지 테스트**
    - [admin.html](file:///d:/Angler/miniGames/games/08-word-search/admin.html)을 엽니다.
    - 난이도를 '쉬움'으로 변경하고 '저장하기'를 누릅니다.
2.  **게임 플레이 테스트**
    - [index.html](file:///d:/Angler/miniGames/games/08-word-search/index.html)로 이동합니다.
    - **격자 크기 확인**: 가로/세로 6칸인지 확인합니다.
    - **타이머 확인**: 3분(03:00)으로 시작하는지 확인합니다.
    - **게임 진행**: 단어를 찾고, 힌트 기능을 사용해 봅니다.
    - **실패/재시도**: 시간을 기다려 타임아웃 화면을 보고, '재시도' 버튼을 눌러 시간이 추가되어 재시작되는지 확인합니다.
