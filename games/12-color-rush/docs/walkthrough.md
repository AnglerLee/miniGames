# 12-color-rush (색깔 스피드) 업데이트 완료

## 변경 사항 요약
사용자의 요청에 따라 UX를 개선하고 관리자 기능을 추가했습니다.

### 1. 관리자(Admin) 페이지 개선
- **설정 항목 추가**:
  - **제한 시간**: 게임의 전체 제한 시간을 설정할 수 있습니다. (기본 60초)
  - **총 문제 수**: 플레이할 문제 수를 설정할 수 있습니다. (기본 15개)
  - **통과 기준**: 클리어에 필요한 최소 정답 수를 설정할 수 있습니다. (기본 12개)
- **저장/로드**: 설정값은 `localStorage`에 저장되어 브라우저를 닫아도 유지됩니다.

### 2. 게임 로직 업데이트
- **전체 시간 제한 모드**: 기존의 무제한 방식에서 긴장감을 주는 카운트다운 방식으로 변경했습니다.
- **재도전 기능**: 시간 초과로 실패 시, "+10초 받고 계속하기" 옵션을 제공하여 사용자가 쉽게 포기하지 않도록 유도했습니다.
- **최고 기록 저장**: 게임 클리어 시, 소요 시간을 바탕으로 최고 기록을 저장하고 시작 화면에 표시합니다.

### 3. UI/UX 디자인 개선
- **타이머 추가**: 화면 상단 중앙에 디지털 타이머를 배치하여 남은 시간을 직관적으로 보여줍니다.
- **통계 표시 변경**: 타이머와 점수판을 나란히 배치하여 가독성을 높였습니다. 정답 카운트는 '현재 정답 / 목표 정답' 으로 표시됩니다.
- **무한 모드 적용**: 정해진 문제 수(15문제) 없이, 제한 시간 내에 목표 점수를 달성하면 성공하는 방식으로 변경되었습니다. (관리자 설정에서 '총 문제 수' 제거됨)
- **재도전 규칙 변경**: 시간 초과 시 '포기하기' 버튼 없이 **'다시 도전 (+2초)'** 버튼만 제공됩니다. 클릭 시 제한 시간을 2초 늘리고 점수를 초기화하여 재시작합니다.
- **오답 페널티**: 정답을 맞추지 못할 경우 남은 시간이 **1초 차감**됩니다.
- **카드 디자인**: 텍스트만 있던 화면을 카드 형태의 깔끔한 디자인으로 변경했습니다.
- **애니메이션**: 문제 등장 시 팝업 애니메이션, 정답/오답 시 피드백 효과를 강화했습니다.
- **버튼 개선**: 터치하기 쉬운 큰 버튼과 색상 그라데이션을 적용했습니다.
