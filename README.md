# 🎮 보물찾기 미니게임 모음

집에서 아이들과 함께 즐기는 보물찾기 게임을 위한 웹 기반 미니게임 모음입니다!

## ✨ 특징

- 📱 **모바일 친화적**: 스마트폰에서 터치로 쉽게 플레이
- 🎯 **20가지 재미있는 게임**: 기억력, 퍼즐, 속도 등 다양한 종류
- 🔐 **비밀번호 시스템**: 게임 클리어 시 비밀번호와 힌트 표시
- 📖 **스토리 모드**: 순차적 게임 진행과 스토리라인
- 🎨 **5가지 테마**: 해적, 우주, 마법, 정글, 비밀요원
- ⚙️ **쉬운 관리**: 웹 브라우저에서 간편하게 설정
- 💾 **서버 불필요**: 모든 데이터는 브라우저에 저장
- 🎨 **예쁜 디자인**: 밝고 친근한 색상과 애니메이션

## 🎮 게임 목록

1. **🌀 미로 탈출** - 공을 움직여 미로를 탈출하는 게임
2. **🃏 짝 맞추기** - 뒤집힌 카드에서 같은 그림 찾기
3. **🔢 숫자 퍼즐** - 슬라이딩 퍼즐로 숫자 정렬하기
4. **➕ 빠른 계산** - 제한 시간 내에 계산 문제 풀기
5. **🔤 단어 찾기** - 격자에서 숨겨진 단어 찾기
6. **🔢 순서대로 터치하기** - 숫자를 순서대로 빠르게 터치
7. **🌈 색깔 스피드** - 색깔 이름과 글자 색 일치 여부 판단

## 🚀 빠른 시작

### ⭐ 두 가지 독립적인 시스템

이 프로젝트는 **두 가지 완전히 분리된 시스템**을 제공합니다:

---

### 🎯 방법 1: 스토리 보물찾기 모드 (추천!)

**순차적인 스토리와 함께 여러 게임을 하나의 모험으로!**

#### 시작하기:
1. `treasure-hunt-admin.html` 파일을 브라우저에서 엽니다
2. "📦 기본 프리셋 로드" 버튼을 클릭하여 5가지 테마 불러오기
3. 원하는 보물찾기를 선택하고 "✅ 활성화" 클릭
4. "🎯 보물찾기 시작" 버튼으로 `treasure-hunt.html` 열기
5. 게임을 순차적으로 완료하며 스토리를 즐기세요!

#### 직접 만들기:
1. `treasure-hunt-admin.html`에서 "➕ 새 보물찾기 만들기" 클릭
2. 이름, 설명, 테마(해적/우주/마법/정글/요원) 선택
3. 게임을 추가하고 드래그하여 순서 조정
4. 각 게임별로:
   - 📖 **스토리 텍스트**: 게임 시작 전 보여줄 이야기
   - 💡 **힌트**: 다음 보물 위치 ("거실 소파 뒤를 찾아봐!")
   - 🔑 **비밀번호**: 게임 완료 시 보여줄 코드
   - 🎉 **성공 메시지**: 클리어 축하 메시지
5. 최종 보상 메시지와 코드 설정
6. 저장 후 활성화!

**특징:**
- 게임 완료 시 자동으로 다음 게임 안내
- 진행 상황 실시간 추적
- 완료한 게임 체크 표시
- 최종 완료 시 특별 보상 화면

---

### 🎮 방법 2: 개별 게임 모드

**각 게임을 독립적으로 플레이하고 관리!**

#### 시작하기:
1. `index.html` 파일을 브라우저에서 엽니다
2. 원하는 게임을 선택하여 플레이합니다

#### 게임별 설정:
1. `admin.html` (전체 게임 관리) 또는 각 게임 폴더의 `admin.html` 열기
2. 각 게임별로 설정:
   - 🔑 **비밀번호/코드**: 클리어 시 보여줄 코드
   - 💡 **힌트 메시지**: 다음 위치 힌트
   - 🎉 **성공 메시지**: 축하 메시지
3. 저장 후 게임 플레이

**특징:**
- 각 게임을 독립적으로 설정 및 관리
- 자유롭게 게임 선택 가능
- 개별 게임 난이도 조정 가능

| 사용 케이스 | 계획된 보물찾기 | 자유 플레이 |

## 📱 모바일에서 사용하기

### 방법 1: QR 코드 (추천)

1. GitHub Pages나 Netlify 등에 업로드
2. QR 코드 생성 사이트에서 게임 URL로 QR 코드 만들기
3. 아이들이 QR 코드를 스캔하여 접속

### 방법 2: 파일 공유

1. 파일을 스마트폰으로 복사
2. 스마트폰 브라우저에서 `index.html` 열기

### 방법 3: 로컬 서버

```bash
# Python 3가 설치되어 있다면
python -m http.server 8000

# 또는 Node.js의 http-server
npx http-server
```

그 후 같은 네트워크의 스마트폰에서 컴퓨터 IP:포트로 접속

## 🎯 사용 시나리오 예시

### 스토리 모드 보물찾기 (신규! 🌟)

1. **준비**: 보물찾기 관리자에서 "해적 보물 찾기" 프리셋 활성화
2. **시작**: 아이에게 태블릿을 주고 `treasure-hunt.html` 열기
3. **진행**: 
   - 게임 1 완료 → 스토리와 힌트 표시 ("거실 소파 뒤를 찾아봐!")
   - 아이가 소파 뒤에서 다음 게임으로 가는 힌트 발견
   - 게임 2 시작 → 완료 → 다음 힌트...
4. **완료**: 모든 게임 클리어 시 최종 보물 위치와 비밀번호 획득!

### 개별 게임 모드 (기존 방식)

1. 집 곳곳에 힌트를 숨깁니다
2. 첫 번째 게임의 힌트를 "소파 뒤"로 설정
3. 아이가 게임 1을 클리어 → 힌트를 보고 소파 뒤에서 종이를 발견
4. 종이에 게임 2의 링크/QR 코드가 있음
5. 게임 2를 클리어 → 다음 힌트 획득
6. 이런 식으로 계속 진행!

### 생일 파티 미션

1. "비밀 요원 미션" 테마 선택
2. 각 게임을 순서대로 진행
3. 마지막 게임의 비밀번호를 생일 선물 상자의 자물쇠 번호로 설정
4. 모든 미션을 완료해야 선물을 열 수 있음!

## ⚙️ 관리자 페이지 기능

### 보물찾기 관리 (treasure-hunt-admin.html) 🌟

**프리셋 관리:**
- 여러 보물찾기 시나리오 생성 및 관리
- 5가지 기본 테마 제공 (해적, 우주, 마법, 정글, 비밀요원)
- 프리셋 활성화/비활성화 (한 번에 하나만)
- 프리셋 복제, 내보내기/가져오기 (JSON)

**게임 구성:**
- 드래그앤드롭으로 게임 순서 조정
- 각 게임별 스토리 텍스트 작성
- 힌트, 비밀번호, 성공 메시지 설정
- 최종 보상 메시지 및 코드 설정

**진행 상황:**
- 현재 진행 중인 보물찾기 확인
- 완료 게임 수 및 진행률 표시

### 개별 게임 관리 (admin.html)

**개별 게임 설정:**
- 각 게임의 설정을 독립적으로 저장
- 테스트 버튼으로 게임 미리보기
- 설정 상태 확인 (설정됨/미설정)

**일괄 작업:**
- **모두 저장**: 모든 게임 설정을 한 번에 저장
- **전체 초기화**: 모든 설정을 기본값으로 되돌리기

## 🐛 문제 해결

### 게임이 저장되지 않아요
- 브라우저의 쿠키/로컬 스토리지가 활성화되어 있는지 확인
- 시크릿 모드에서는 저장이 안 될 수 있음

### 모바일에서 터치가 안 돼요
- 화면을 길게 눌러보세요 (일부 게임은 드래그 필요)
- 브라우저를 최신 버전으로 업데이트

### 비밀번호가 표시되지 않아요
- 관리자 페이지에서 저장했는지 확인
- 같은 브라우저를 사용하고 있는지 확인
