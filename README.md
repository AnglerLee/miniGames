# 🎮 보물찾기 미니게임 모음

집에서 아이들과 함께 즐기는 보물찾기 게임을 위한 웹 기반 미니게임 모음입니다!

## ✨ 특징

- 📱 **모바일 친화적**: 스마트폰에서 터치로 쉽게 플레이
- 🎯 **20가지 재미있는 게임**: 기억력, 퍼즐, 속도 등 다양한 종류
- 🏠 **홈 트레저헌트**: 디지털 + 실물 보물찾기 하이브리드 시스템
- 🎨 **9가지 테마**: 미스터리, 해적, 우주, 마법, 공룡, 정글, 바다, 닌자, 유령의집
- 👥 **1~2인 모드**: 혼자 또는 둘이서 번갈아 플레이
- 🔐 **관리자 PIN**: 4자리 PIN으로 설정 보호
- 💾 **서버 불필요**: 모든 데이터는 브라우저 localStorage에 저장
- 🎊 **풍부한 효과**: Confetti, 사운드, 진동 피드백

## 🏠 홈 트레저헌트 (treasure-hunt3)

디지털 미니게임과 실물 보물찾기를 결합한 하이브리드 보물찾기 시스템입니다.

### 핵심 컨셉

1. 아이가 **미니게임 미션**을 클리어하면 **단서(힌트)** 를 획득
2. 단서를 조합하여 **집 안 어딘가에 숨겨진 보물**의 위치를 추리
3. 보물을 찾으면 **비밀번호**를 입력하여 보물상자를 오픈!

### 아키텍처

```
index.html ──── #main (시작 화면)
    │           #intro (스토리 인트로)
    │           #hub  (미션 허브)
    │
    ├── play.html?type=mini&idx=0  ── 미니게임 (iframe)
    ├── play.html?type=find&idx=1  ── 물건찾기 미션
    │
    └── treasure.html ─────────────── 보물상자 (비밀번호 입력)

admin.html ──── PIN 인증 → 대시보드 (설정 관리)
```

### 페이지별 기능

| 페이지 | 역할 | 주요 기능 |
|--------|------|-----------|
| `index.html` | 메인 & 허브 | 시작 화면, 스토리 인트로, 미션 목록, 진행률, 단서 모음 |
| `admin.html` | 관리자 설정 | PIN 인증, 테마/시나리오 선택, 미션 구성, 옵션 설정 |
| `play.html` | 미션 플레이 | 미니게임 iframe 로드, 물건찾기(힌트+정답), 클리어 오버레이 |
| `treasure.html` | 보물상자 | 3D 보물상자 애니메이션, 비밀번호 입력, 결과 & 성적 표시 |

### 9가지 테마

| 테마 | ID | 색상 톤 |
|------|----|---------|
| 🔍 미스터리 탐정 | `mystery` | 다크 블루 + 골드 |
| 🏴‍☠️ 해적 보물 | `pirate` | 브라운 + 골드 |
| 🚀 우주 탐험 | `space` | 딥 블루 + 시안 |
| 🧙 마법 학교 | `magic` | 퍼플 + 핑크 |
| 🦕 공룡 발굴 | `dino` | 그린 + 오렌지 |
| 🌴 정글 탐험 | `jungle` | 다크 그린 + 라임 |
| 🐠 심해 탐험 | `ocean` | 네이비 + 시안 |
| 🥷 닌자 수련 | `ninja` | 다크 그레이 + 레드 |
| 👻 유령의 집 | `haunted` | 퍼플 + 퍼플 |

각 테마에는 2개 이상의 프리셋 시나리오가 포함되어 있어 바로 플레이할 수 있습니다.

### 미션 타입

**🎮 미니게임 미션 (`minigame`)**
- 20개 미니게임 중 선택하여 iframe으로 로드
- postMessage 통신으로 클리어 감지
- 2인 모드 시 플레이어가 번갈아 플레이

**🔎 물건찾기 미션 (`findObject`)**
- 집 안에서 실물을 찾는 미션
- 최대 3개 힌트 (단계적 공개, 3회 실패 시 전체 공개)
- 정답 입력 (대소문자/공백 무시)
- 2인 모드 시 함께 협력 플레이

### 관리자 옵션

| 옵션 | 설명 |
|------|------|
| 테마 선택 | 9개 테마 중 선택 |
| 시나리오 선택 | 테마별 프리셋 시나리오 로드 |
| 미션 구성 | 미니게임/물건찾기 추가, 순서 변경, 삭제 |
| 플레이어 수 | 1인 / 2인 모드 (이름 설정) |
| 시간 제한 | 전체 게임 시간 제한 (분 단위, OFF 가능) |
| 보물 비밀번호 | 최종 보물상자 오픈 비밀번호 |
| 비밀 코드 | 보물상자 오픈 후 표시되는 코드 |
| 단서 표시 | 허브 화면에서 수집한 단서 표시/숨김 토글 |
| 관리자 PIN | 4자리 PIN 변경 (기본: 0000) |

### localStorage 키

| 키 | 용도 |
|----|------|
| `treasureHunt3_config` | 게임 설정 (테마, 미션, 옵션 등) |

| `treasureHunt3_admin` | 관리자 PIN |
| `treasureHunt3_history` | 플레이 기록 (최대 50건) |

## 🚀 빠른 시작

### 🏠 홈 트레저헌트 (추천!)

1. `treasure-hunt3/admin.html`을 브라우저에서 엽니다
2. PIN 입력 (기본: `0000`)
3. 테마를 선택하고 시나리오를 불러옵니다
4. 미션을 원하는 대로 커스터마이징합니다
5. 보물 비밀번호를 설정하고 저장합니다
6. 집 안 곳곳에 실물 힌트를 숨깁니다
7. `treasure-hunt3/index.html`을 열어 게임을 시작합니다!

### 🎮 개별 게임 모드

1. `index.html`을 브라우저에서 엽니다
2. 원하는 게임을 선택하여 플레이합니다
3. 각 게임의 `admin.html`에서 비밀번호/힌트 설정 가능

## 🎯 사용 시나리오 예시

### 생일 파티 보물찾기

1. **준비**: admin에서 "마법 학교" 테마 선택 → 시나리오 로드
2. **커스터마이징**: 물건찾기 미션의 힌트를 집 구조에 맞게 수정
3. **보물 숨기기**: 비밀번호 `생일축하해`로 설정, 선물과 함께 숨기기
4. **시작**: 아이에게 스마트폰을 주고 게임 시작!
5. **진행**: 미니게임 클리어 → 단서 획득 → 물건찾기 → 다음 미션
6. **피날레**: 모든 미션 완료 → 단서 조합 → 보물 발견 → 비밀번호 입력 → 🎉

### 주말 가족 활동

1. **2인 모드** 설정 → 형제/자매가 번갈아 미니게임 플레이
2. 물건찾기 미션은 함께 협력하여 진행
3. 최종 성적으로 실력 비교!

## 📱 모바일에서 사용하기

### 방법 1: 로컬 서버

```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server
```

같은 Wi-Fi 네트워크의 스마트폰에서 `컴퓨터IP:8000`으로 접속

### 방법 2: GitHub Pages (무료)

1. GitHub에 코드 업로드
2. Settings → Pages → Source를 main branch로 설정
3. 생성된 URL로 어디서든 접속 가능

### 방법 3: 파일 직접 전송

스마트폰으로 파일 복사 후 브라우저에서 `index.html` 열기

## 💾 데이터 저장

모든 설정과 진행 상태는 브라우저의 `localStorage`에 저장됩니다:
- 서버가 필요 없음
- 같은 브라우저에서는 새로고침해도 유지
- 다른 브라우저나 기기에서는 별도 설정 필요

## 🐛 문제 해결

| 문제 | 해결 방법 |
|------|-----------|
| 게임이 저장되지 않음 | 브라우저 localStorage 활성화 확인, 시크릿 모드 X |
| 모바일 터치 안 됨 | 브라우저 최신 버전 업데이트 |
| 비밀번호 표시 안 됨 | admin에서 저장 확인, 같은 브라우저 사용 확인 |
| PIN을 잊어버림 | 브라우저 개발자도구 → Console → `localStorage.removeItem('treasureHunt3_admin')` |
