# 🎮 보물찾기 미니게임 모음

집에서 아이들과 함께 즐기는 보물찾기 게임을 위한 웹 기반 미니게임 모음입니다!

## ✨ 특징

- 📱 **모바일 친화적**: 스마트폰에서 터치로 쉽게 플레이
- 🎯 **7가지 재미있는 게임**: 기억력, 퍼즐, 속도 등 다양한 종류
- 🔐 **비밀번호 시스템**: 게임 클리어 시 비밀번호와 힌트 표시
- ⚙️ **쉬운 관리**: 웹 브라우저에서 간편하게 설정
- 💾 **서버 불필요**: 모든 데이터는 브라우저에 저장
- 🎨 **예쁜 디자인**: 밝고 친근한 색상과 애니메이션

## 🎮 게임 목록

1. **🌀 미로 탈출** - 공을 움직여 미로를 탈출하는 게임
2. **🃏 짝 맞추기** - 뒤집힌 카드에서 같은 그림 찾기
3. **🔢 숫자 퍼즐** - 슬라이딩 퍼즐로 숫자 정렬하기
4. **➕ 빠른 계산** - 제한 시간 내에 계산 문제 풀기
5. **🔤 단어 찾기** - 격자에서 숨겨진 단어 찾기
6. **🔢 순서대로 터치하기** - 숫자를 순서대로 빠르게 터치
7. **🌈 색깔 스피드** - 색깔 이름과 글자 색 일치 여부 판단

## 🚀 빠른 시작

### 1. 파일 다운로드

이 폴더의 모든 파일을 컴퓨터에 다운로드하세요.

### 2. 게임 설정

1. `admin.html` 파일을 브라우저에서 엽니다
2. 각 게임별로 비밀번호와 힌트를 입력합니다
   - **비밀번호**: 게임 클리어 시 보여줄 코드 (예: "1234", "소파뒤")
   - **힌트 메시지**: 다음 보물 위치 힌트 (예: "거실 소파 뒤를 찾아봐!")
   - **성공 메시지**: 축하 메시지 (예: "잘했어! 다음 미션으로 가자!")
3. "모두 저장" 버튼을 클릭합니다

### 3. 게임 시작

1. `index.html` 파일을 브라우저에서 엽니다
2. 아이들에게 보여주고 게임을 선택하게 합니다
3. 게임을 클리어하면 비밀번호/힌트가 표시됩니다!

## 📱 모바일에서 사용하기

### 방법 1: QR 코드 (추천)

1. GitHub Pages나 Netlify 등에 업로드
2. QR 코드 생성 사이트에서 게임 URL로 QR 코드 만들기
3. 아이들이 QR 코드를 스캔하여 접속

### 방법 2: 파일 공유

1. 파일을 스마트폰으로 복사
2. 스마트폰 브라우저에서 `index.html` 열기

### 방법 3: 로컬 서버

```bash
# Python 3가 설치되어 있다면
python -m http.server 8000

# 또는 Node.js의 http-server
npx http-server
```

그 후 같은 네트워크의 스마트폰에서 컴퓨터 IP:포트로 접속

## 🎯 사용 시나리오 예시

### 집안 보물찾기

1. 집 곳곳에 힌트를 숨깁니다
2. 첫 번째 게임의 힌트를 "소파 뒤"로 설정
3. 아이가 게임 1을 클리어 → 힌트를 보고 소파 뒤에서 종이를 발견
4. 종이에 게임 2의 링크/QR 코드가 있음
5. 게임 2를 클리어 → 다음 힌트 획득
6. 이런 식으로 계속 진행!

### 생일 파티 미션

1. 각 게임을 순서대로 진행
2. 마지막 게임의 비밀번호를 생일 선물 상자의 자물쇠 번호로 설정
3. 모든 게임을 클리어해야 선물을 열 수 있음!

## ⚙️ 관리자 페이지 기능

### 개별 게임 관리
- 각 게임의 설정을 독립적으로 저장
- 테스트 버튼으로 게임 미리보기
- 설정 상태 확인 (설정됨/미설정)

### 일괄 작업
- **모두 저장**: 모든 게임 설정을 한 번에 저장
- **전체 초기화**: 모든 설정을 기본값으로 되돌리기

### 설정 예시

**게임 1: 미로 탈출**
- 비밀번호: `빨간상자`
- 힌트: `거실 TV 옆 빨간 상자를 찾아봐!`
- 성공 메시지: `대단해! 다음 미션을 찾아가자!`

## 🔧 커스터마이징

### 색상 테마 변경

`assets/css/common.css` 파일의 `:root` 섹션에서 색상 변경:

```css
:root {
    --primary-color: #4CAF50;  /* 메인 색상 */
    --secondary-color: #2196F3; /* 보조 색상 */
    /* ... */
}
```

## 🌐 온라인 호스팅

### GitHub Pages (무료)

1. GitHub 계정 생성
2. 새 repository 생성
3. 파일들을 업로드
4. Settings → Pages → Source를 main branch로 설정
5. 생성된 URL로 접속 가능

### Netlify (무료)

1. [Netlify](https://netlify.com) 가입
2. "New site from Git" 또는 폴더 드래그 앤 드롭
3. 자동으로 배포되고 URL 생성

## 🐛 문제 해결

### 게임이 저장되지 않아요
- 브라우저의 쿠키/로컬 스토리지가 활성화되어 있는지 확인
- 시크릿 모드에서는 저장이 안 될 수 있음

### 모바일에서 터치가 안 돼요
- 화면을 길게 눌러보세요 (일부 게임은 드래그 필요)
- 브라우저를 최신 버전으로 업데이트

### 비밀번호가 표시되지 않아요
- 관리자 페이지에서 저장했는지 확인
- 같은 브라우저를 사용하고 있는지 확인
