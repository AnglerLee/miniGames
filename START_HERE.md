# 🎮 어디서 시작할까요?

이 프로젝트는 **두 가지 완전히 독립적인 방식**으로 사용할 수 있습니다!

---

## 🎯 스토리 보물찾기 (추천!)

**연결된 스토리와 함께 순차적으로 게임을 진행하는 모험!**

### 시작 방법:
1. 📂 `treasure-hunt-admin.html` 열기
2. 🎨 "기본 프리셋 로드" → 5가지 테마 중 선택
3. ✅ 원하는 보물찾기 "활성화"
4. 🎯 "보물찾기 시작" 버튼 클릭
5. 🎮 게임 시작!

### 이런 분들께 추천:
- 🏠 집안 보물찾기 이벤트를 준비하는 분
- 📖 스토리가 있는 게임을 원하는 분
- 🎉 특별한 날(생일, 기념일) 이벤트
- 👨‍👩‍👧‍👦 아이들과 계획된 활동

---

## 🎮 개별 게임 플레이

**각 게임을 자유롭게 선택하여 플레이!**

### 시작 방법:
1. 📂 `index.html` 열기
2. 🎮 원하는 게임 선택
3. 🎯 바로 플레이!

### 게임 설정 (선택사항):
- 📂 `admin.html` 열기
- ⚙️ 각 게임별 힌트/비밀번호 설정
- 💾 저장

### 이런 분들께 추천:
- 🎲 자유롭게 게임을 즐기고 싶은 분
- 🎯 특정 게임만 플레이하고 싶은 분
- 🔄 반복해서 게임을 즐기는 분
- 📱 간단한 미니게임을 원하는 분

---

## 🤔 어떤 것을 선택해야 할까요?

### 🎯 스토리 보물찾기를 선택하세요:
- ✅ 집안에서 보물찾기 이벤트를 하고 싶다
- ✅ 아이들에게 스토리가 있는 미션을 주고 싶다
- ✅ 5개 이상의 게임을 순서대로 진행하고 싶다
- ✅ 각 게임마다 다른 힌트를 주고 싶다

### 🎮 개별 게임을 선택하세요:
- ✅ 그냥 재미있는 게임을 하고 싶다
- ✅ 특정 게임만 반복해서 하고 싶다
- ✅ 간단하게 시작하고 싶다
- ✅ 자유롭게 게임을 선택하고 싶다

## 🚀 빠른 실행

```bash
# 방법 1: Python이 설치되어 있다면
python -m http.server 8000

# 방법 2: Node.js가 설치되어 있다면
npx http-server

# 그 후 브라우저에서 http://localhost:8000 접속
```

또는 그냥 HTML 파일을 더블클릭하여 브라우저로 열어도 됩니다!
